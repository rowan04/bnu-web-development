Term Definition
Agile A software development methodology that values having a dynamic and flexible plan. Agile methods have a strong emphasis on communication, teamwork, leadership and project management
AJAX (Asynchronous JavaScript and XML) - A Programmatic approach used to develop highly interactive and responsive web applications by exploiting a set of technologies. Web development techniques that use XML, HTML, JavaScript and CSS for web service interaction allowing for faster and more interactive web applications
Angular A typescript-based web application framework and platform that supports full-stack development for building single-page applications. It is an open-source project maintained by Google
Apache A cross-platform software that enables a computer to host one or more websites that can be accessed over the internet using a web browser
Bootstrap A free, open source front-end development framework for the creation of websites and web apps. Designed to enable responsive development of mobile-first websites. To boot or to load a program into a computer using a much smaller initial program to load in the desired program, which is usually an OS
CSS (Cascading style sheets) - A language which is used to style the content of web pages. The CSS tells the browser how each HTML element should appear
Cloud A global network of servers that are accessed over the internet. The cloud provides on-demand access to computing resources, such as applications, servers, data storage, development tools and networking capabilities
DHTML (Dynamic Hypertext Markup Language) - The combination of HTML, style sheets and client-side scripts that enabled the creation of interactive and animated documents
DNS (Domain Name System) - Responsible for translating domain names into a specific IP address so that the initiating client can load the requested internet resources
DOM (Document Object Model) - An application programming interface (API) for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated
