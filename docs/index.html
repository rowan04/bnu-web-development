Term Definition
Agile A software development methodology that values having a dynamic and flexible plan. Agile methods have a strong emphasis on communication, teamwork, leadership and project management
AJAX (Asynchronous JavaScript and XML) - A Programmatic approach used to develop highly interactive and responsive web applications by exploiting a set of technologies. Web development techniques that use XML, HTML, JavaScript and CSS for web service interaction allowing for faster and more interactive web applications
Angular A typescript-based web application framework and platform that supports full-stack development for building single-page applications. It is an open-source project maintained by Google
Apache A cross-platform software that enables a computer to host one or more websites that can be accessed over the internet using a web browser
Bootstrap A free, open source front-end development framework for the creation of websites and web apps. Designed to enable responsive development of mobile-first websites
CSS (Cascading style sheets) - A language which is used to style the content of web pages. The CSS tells the browser how each HTML element should appear
Cloud A global network of servers that are accessed over the internet. The cloud provides on-demand access to computing resources, such as applications, servers, data storage, development tools and networking capabilities
DHTML (Dynamic Hypertext Markup Language) - The combination of HTML, style sheets and client-side scripts that enabled the creation of interactive and animated documents
DNS (Domain Name System) - Responsible for translating domain names into a specific IP address so that the initiating client can load the requested internet resources
DOM (Document Object Model) - An application programming interface (API) for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated
Firewall A network security device that inspects incoming and outgoing traffic and decides whether to allow or block it based on a predefined set of rules
FTP (File Transfer Protocol) - A network communication protocol that enables the delivery of digital files from a server to the client
GDPR (General Data Protection Regulation) - A law designed to protect personal data stored on computers or in an organised paper filing system
GIF (Graphics Interchange Format) - A digital file format devised as a means of reducing the size of images and short animations
Git Git is a distributed version control system that tracks changes in any set of computer files. Git is used for coordinating work among programmers collaboratively developing source code during software development
GitHub A web-based platform that uses Git to host and manage software projects. It allows developers to collaborate on open source projects and share files
HTML (Hypertext Markup Language) - A standardised markup language used to format all web pages on the WWW. HTML elements are the building blocks of a web page
HTTP (Hypertext Transfer Protocol) - The underlying protocol used by the WWW. It defines how messages (files and data) are formatted and transmitted
Java A high-level, object-oriented programming language that can run on multiple platforms. It is designed to have as few implementation dependencies as possible
JavaScript A client-side scripting language, which means the source code is processed by the client's web browser rather than on the web server. This means JavaScript functions can run after a webpage has loaded without communicating with the server
JPEG (Joint Photographic Experts Group) - A digital image compression format, and a popular image type shown on the Internet and used with digital cameras. Uses lossy compression
JSON (JavaScript Object Notation) - A standard text-based format used to portray structured data. Developed from the JavaScript object syntax, but grew out of it
jQuery A JavaScript library that allows web developers to add extra functionality to their websites. It is open source and free under the MIT license
KML (Keyhole Markup Language) - An XML notation for expressing geographic annotation and visualisation within 2D maps and 3D Earth browsers
PHP (Hypertext Preprocessor) - A server-side scripting language like ASP that supports various databases like MySQL and Oracle. It is open-source
PNG (Portable Network Graphics) - A file format used for lossless image compression. A standard format for web graphics
RSS (Really Simple Syndication) - XML formatted text commonly used for distributing weblogs, news or other content that is frequently updated. Allows users and applications to access updates to websites in a computer-readable format
SEO (Search Engine Optimization) - The practice of modifying the design and content of a website to achieve visibility online
SSL (Secure Sockets Layer) - A computer networking procedure for safeguarding connections between network application clients and servers over a vulnerable network, like the internet
SVG (Scalable Vector Graphics) - An XML based vector image format for defining two-dimensional graphics, having support for interactivity and animation
TCP/IP (Transmission Control Protocol / Internet Protocol) - The language a computer uses to access the internet. It consists of a suite of protocols designed to establish a network of networks to provide a host with access to the internet
UDP (User Datagram Protocol ) - A data transport protocol within the IP suite. A connectionless protocol that doesn't rely on two hosts establishing a connection before transferring datagrams
URI (Uniform Resource Identifier) - A sequence of characters identifying a hypertext resource
URL (Uniform Resource Locator) - An address that specifies the location and protocol for accessing a resource on the internet or a local network
UX (User Experience) - A concept in computing system and application design that studies and evaluates human feelings and expressions when using such systems
Web Accessibility The design and development of websites, tools and technologies that are usable by people with disabilities
Wiki An online hypertext publication collaboratively edited and managed by its own audience, using a web browser
W3C (World Wide Web Consortium) - An international organisation that creates standards for the World Wide Web
WinSCP (Windows Secure Copy) - A free and open-source client for secure file transfer between a local computer and a remote server
XML (Extensible Markup Language) - A markup language that defines a set of rules for encoding documents in a format that is both human and machine readable
XSL (Extensible Stylesheet Language) - Refers to a family of languages used to transform and render XML documents
